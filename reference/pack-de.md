## Localized set & card names — use the DE name, never the English alone
On a German post the English-only set name reads as someone who doesn't follow the local releases.
Verify the localized name at run time (pokemon.com DE / PokéWiki / Bisafans). If the English name is
widely known, "Localized (English)" is a fine bridge.
- **ME05 "Pitch Black" = "Dunkelnacht"** — full form **"Mega-Entwicklung – Dunkelnacht"**.
  - ✅ "Mega-Entwicklung – Dunkelnacht", or "Dunkelnacht", or "Dunkelnacht (Pitch Black)"
  - ❌ "Pitch Black" alone · ❌ "Nuit Noire" (that's the FRENCH name — never on a DE post)
  - **Orthography:** the series prefix uses an **en dash** — "Mega-Entwicklung – Dunkelnacht", not a hyphen.
- **ME04 "Chaos Rising" = "Wachsendes Chaos"** (chase: **Mega-Quajutsu-ex**). To a German audience
  "Wachsendes Chaos" already MEANS the German version — never say "das deutsche Wachsendes Chaos".
  For another language, name it explicitly: the EN print is **"Chaos Rising"**, the JP base is
  **"Ninja Spinner"**.
- **Pokémon names (DE):** Glurak (Charizard), Quajutsu (Greninja), Evoli (Eevee), Relaxo (Snorlax),
  Mewtu (Mewtwo). Look up any other at run time. (Full list in `slang-de.md`.)
- Set facts (dates, chase cards, counts) are language-neutral → see `verified-facts.md`.

## Approved DE diction — the English-German hybrid is native, don't "correct" it
German TCG collectors mix English loanwords heavily; that mix is what reads native. Keep words like
**"rippen", "Hold-Liste", "Upside", "Standard-Ware", "Pull", "Chase", "sealed", "Drop", "Restock",
"Single", "Alt Art"** — do not translate them to pure German. Use *du*, never *Sie*.

## One term per concept (DE pairs to watch)
Once you pick a word on slide 1, repeat it on every later slide — don't alternate synonyms.
In German specifically, don't mix **"sealed" / "versiegelt"** (pick one and keep it). (The universal
consistency rule is in core; this is the DE pair it most often trips on.)

## DE-specific factual cautions
- **Don't blame "Überproduktion" for flat ETB value** — the 2024–2026 market was defined by
  shortages and scalping, not oversupply, so the overproduction framing reads as out-of-touch. To make
  the "standard ETBs don't appreciate" point, cite a concrete older example instead.
  - ✅ "alte SV-ETBs, die jahrelang in jedem Müller liegen" · "Standard-ETBs ohne Limitierung"
  - ❌ "massig gedruckte Standard-ETBs — bei Überproduktion bleibt der Wert flat"
  - Note: **"Müller" is a German retailer** — it belongs only on DE posts (the FR equivalent is a
    French chain, verified at run time).

## Banned promo words (DE) — never on a slide or in a caption
**"kauf jetzt", "Rabattcode", "bestes Shop", "Werbung"** (and the ad-smell list in `slang-de.md`:
"unschlagbares Angebot", "jetzt zugreifen", "garantiert günstigster Preis"). They turn a collector
post into an ad. (The *principle* — no overt promo / ad-speak — is universal and lives in core; these
literal words are the DE instances.)
