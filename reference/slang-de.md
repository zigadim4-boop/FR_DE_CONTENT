# German Pokémon TCG — Community Lexicon & Tone

How German collectors actually talk. The DE community mixes **English loanwords** heavily — using them is what makes you sound native, not cringe. Lea speaks this way naturally. Use *du*, never *Sie*.

## Products & sealed (the only thing IvoryShard sells)
| Term | Meaning / usage |
|---|---|
| **Display** | Booster box (36 / 18 packs). "Ein Display Wachsendes Chaos." The classic box. |
| **Booster** | Single pack. "Ich mach einen Booster auf." |
| **Blister / Klemmblister** | Small sealed blister pack(s), often w/ promo. |
| **ETB / Top-Trainer-Box (TTB)** | Elite Trainer Box. Most say **"ETB"**; official DE name is *Top-Trainer-Box*. |
| **Tin** | Metal tin. Said in English. |
| **Booster Bundle / Bundle** | 6-pack bundle. English. |
| **Sammelkollektion / Premium Kollektion** | Collection boxes. |
| **sealed / versiegelt** | Sealed. "Sealed lassen", "bleibt sealed." *sealed* is more common spoken than *versiegelt*. |
| **Vorbestellung / Preorder** | Preorder. |

## Cards, pulls, rarity
| Term | Meaning / usage |
|---|---|
| **Pull** | What you pull. "Krasser Pull", "ich hab eine Alt gepullt." Verb: *pullen / ziehen*. |
| **ziehen** | To pull/draw. "Was hast du gezogen?" |
| **Chase / Chase Card** | The set's most-wanted card. English. |
| **Single** | Single card (bought individually). |
| **Alt / Alt Art / Sonderillustration (SIR)** | Alternate art. **"Alt Art"** spoken; official DE: *Sonderillustration* / *Illustrationsrarität*. |
| **Full Art / FA** | Full-art card. |
| **Gold / Goldkarte** | Gold secret rare. |
| **No Pull / Griff ins Klo** | Bad pack/box. |
| **Banger** | Great pull. "Was'n Banger." |
| **Marktwert / Cardmarket-Preis** | Market value. "Schau den Marktwert vor dem Kauf." |
| **gegradet / PSA** | Graded. "PSA 10", "gegradete Karte." |
| **Brick** | A product that loses value / never appreciates. English, used as-is. |

## The anti-scalper vocabulary (important — IvoryShard's positioning)
| Term | Meaning |
|---|---|
| **Scalper / Wiederverkäufer** | Scalper / reseller (negative). |
| **Mogelpackung** | A "cheated" pack — repackaged/searched. |
| **Wiegen / gewogene Booster** | Weighing packs (scalper tactic to find hits) — collectors hate it. |
| **Checkout-Limits** | Purchase limits per customer = fair drops. IvoryShard USP. |
| **wöchentliche Drops** | Weekly drops. "So bekommt jeder sein Stück." |
| **Restock-Benachrichtigungen** | Restock notifications. Hero CTA hook. |

## Pokémon names (DE) — use DE names with the DE audience
- **Glurak** = Charizard (the king / biggest chase magnet)
- **Pikachu** = Pikachu · **Evoli** = Eevee · **Mew** = Mew · **Mewtu** = Mewtwo
- **Quajutsu** = Greninja (Mega-Quajutsu-ex = Wachsendes Chaos chase)
- **Relaxo** = Snorlax · **Schiggy** = Squirtle · **Glumanda** = Charmander · **Bisasam** = Bulbasaur
- **Mega-Entwicklung / Mega-[Name]** = Mega Evolution (current era)

## Sets / eras (DE names)
- **Karmesin & Purpur (KP)** = Scarlet & Violet era
- **Mega-Entwicklung** = current Mega Evolution era
- **Wachsendes Chaos** = the GERMAN-language Chaos Rising (ME04, Release 22.05.2026). Chase: **Mega-Quajutsu-ex**.
  - ⚠️ **Naming rule:** to a German audience, "Wachsendes Chaos" already MEANS the German version — never say "das deutsche Wachsendes Chaos" (redundant/odd). If you mean a non-German version, name it explicitly and naturally: the English print is called **"Chaos Rising"** (collectors use the English set name for the EN version), e.g. "die englische Version / Chaos Rising". Same logic for other languages ("die japanische Version").
- **Erhabene Helden** = Ascended Heroes (ME2.5, 30.01.2026)
- **151** = Kanto set (very DE-collected)
- **Prismatische Entwicklungen** = Prismatic Evolutions

## Tone notes — sounds native vs sounds like an ad
- ✅ Native: *"ehrlich", "am Ende", "lohnt sich", "krass", "mega", "auf jeden Fall", "kann passieren", "fühlt sich an wie Community", "spar dir das"*
- ❌ Ad-smell: *"unschlagbares Angebot", "jetzt zugreifen", "garantiert günstigster Preis"*, ❗❗❗, all-caps selling, Werbe-Deutsch.
- Use *du*. Mix in English loanwords (*Pull, Chase, sealed, Drop, Restock, Single, Alt Art*) — it's how real DE collectors talk.
