## Localized set & card names — use the FR name, never the English alone
On a French post the English-only set name reads as someone who doesn't follow the local releases.
Verify the localized name at run time (pokemon.com FR / Poképédia / Pokécardex). If the English name
is widely known, "Localized (English)" is a fine bridge.
- **ME05 "Pitch Black" = "Nuit Noire"** — full form **"Méga-Évolution – Nuit Noire"** (verified
  2026-06-09 via pokemon.com FR / press.pokemon.com FR). Headliner cards **Méga-Darkrai-ex** and
  **Méga-Zeraora-ex** (hyphenated FR form).
  - ✅ "Méga-Évolution – Nuit Noire", or "Nuit Noire", or "Nuit Noire (Pitch Black)"
  - ❌ "Pitch Black" alone · ⚠️ ❌ "Dunkelnacht" (that's the GERMAN name — never on a FR post), and
    never German date phrasing ("Erscheint am…") on a French post.
- **ME04 "Chaos Rising" = "Chaos Ascendant"** (chase: **Méga-Amphinobi-ex**). The JP base is
  **"Ninja Spinner"** — never the FR/EN name.
- **French Pokémon names (never the English name):** **Amphinobi** (Greninja), **Gallame** (Gallade),
  **Crocorible** (Krookodile), Dracaufeu (Charizard), Évoli (Eevee), Ronflex (Snorlax). Look up any
  other at run time. (Full list in `slang-fr.md`.)
- **Méga ex formatting:** prefer the official hyphenated prose form **"Méga-Amphinobi-ex"** (used by
  pokemon.com France); marketplaces also write **"Méga-Amphinobi ex"** with a space — both are seen,
  so don't claim the hyphenated form is the only correct one.
- **"A-Z"** (the character in AZ's Tranquility) is spelled with the hyphen on FR posts — do NOT change
  it to "AZ". Look up the official FR card name at run time.
- Set facts (dates, chase cards, counts) are language-neutral → see `verified-facts.md`.

## Native FR diction — and the Anglicisms to avoid
Use the genuinely native terms; the FR community uses English loanwords heavily, but prefer the French
equivalent where one is native (verified 2026-06-04 via Poképédia / Pokécardex). Use *tu*, never *vous*;
contractions always.
- ✅ native FR: **"scellé"** (sealed), **"display"**, **"la cote"** (market value).
- ⚠️ prefer the FR term over the Anglicism: **"carte à l'unité"** > "single"; **"Banger" / "carte
  chase"** > bare "chase". **"rip / ripper"** is informal English-borrowed streamer slang — use
  sparingly, not as standard French.
- ⚠️ when pointing to where you buy, prefer **"là-bas"** or **"chez eux"** over a bare **"là"** —
  a standalone "là" is clunky ("je le prends en scellé chez eux", not "…en scellé là"). [FR 2026-06-07]

## One term per concept (FR pair to watch)
Once you pick a word on slide 1, repeat it on every later slide. In French specifically, don't mix
**"sealed" / "scellé"** — pick "scellé" and keep it. (The universal consistency rule is in core; this
is the FR pair it most often trips on.)

## Proofread (FR)
Typos kill credibility instantly — proofread every French word before it ships.
- ❌ "partou" → ✅ "partout"

## Banned promo words (FR) — never on a slide or in a caption
**"code promo", "-10%", "le meilleur site", "achetez", "sponso"** (and the ad-smell list in
`slang-fr.md`: "profitez de notre offre exceptionnelle", "le meilleur prix garanti", "n'attendez
plus"). They turn a collector post into an ad. (The *principle* — no overt promo / ad-speak — is
universal and lives in core; these literal words are the FR instances.)
