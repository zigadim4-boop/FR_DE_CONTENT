# French Pokémon TCG — Community Lexicon & Tone

How French collectors actually talk. The FR community uses **a lot of English loanwords** — using them is what makes you sound native; over-translating sounds like a brand. Chloé speaks this way naturally.

## Products & sealed (the only thing IvoryShard sells)
| Term | Meaning / usage |
|---|---|
| **Display** | Booster box (36 boosters, or 18 for newer SV sets). "Un display de Chaos Ascendant." Most common pull-y purchase. |
| **Booster** | A single pack. "J'ouvre un booster." |
| **Blister** | Small sealed pack(s) on cardboard, often w/ a promo. "Blister x3." |
| **ETB / Coffret Dresseur d'Élite** | Elite Trainer Box. Most say **"ETB"** out loud; "coffret dresseur d'élite" is the official FR name. |
| **Pokébox / Tin** | Metal tin collection. |
| **Coffret** | Any boxed collection (Premium, Coffret Collection, etc.). |
| **Bundle / Booster Bundle** | 6-pack bundle. Said in English. |
| **Scellé** | Sealed. "Acheter du scellé", "rester scellé" (keep sealed). Opposite of single. |
| **Précommande** | Preorder. |

## Cards, pulls, rarity
| Term | Meaning / usage |
|---|---|
| **Pull** | What you pull from a pack. "Gros pull", "j'ai pull une alt." Verb: *pull / pull une carte*. |
| **Chase / carte chase** | The card everyone wants from a set. |
| **Single** | A card bought individually (not from a pack). "Acheter en single." |
| **Alt / Alt art / Illustration spéciale (IS)** | Alternate-art card. **"Alt art"** is most common spoken. Official FR: *illustration spéciale*. |
| **SIR / Illustration rare spéciale** | Special Illustration Rare. Usually said "SIR" or "alt". |
| **Full art / FA** | Full-art card. |
| **Gold / Carte or** | Gold secret rare. |
| **Rainbow** | Rainbow rare (older eras). |
| **God pack** | A pack with all hits (rare). |
| **No pull / carton** | Bad pack/box, nothing good. "Display no pull." |
| **Banger** | A great pull. "Quel banger." |
| **La cote** | The market value/price of a card. "Regarde la cote avant d'acheter." (Cardmarket trend price.) |
| **Gradée / PSA / gradation** | Graded card. "PSA 10", "carte gradée." |
| **Brick / un four** | A product whose value tanks / never goes up. Also "ça va faire un four." |

## Pokémon names (FR) — use FR names with the FR audience
- **Dracaufeu** = Charizard (the king, biggest chase magnet)
- **Pikachu** = Pikachu
- **Évoli** = Eevee · **Mew** = Mew · **Mewtwo** = Mewtwo
- **Amphinobi** = Greninja (Méga-Amphinobi-ex = Chaos Ascendant chase). ⚠️ *Quajutsu* is the GERMAN name — never use it on the FR account.
- **Ronflex** = Snorlax · **Carapuce** = Squirtle · **Salamèche** = Charmander · **Bulbizarre** = Bulbasaur
- **Méga-Évolution / Méga-[nom]** = Mega Evolution (current era)
- Tip: when in doubt about a FR name, use it but you can gloss with the EN/JP name for clarity in a comparison post.

## Sets / eras (FR names)
- **Écarlate & Violet (EV)** = Scarlet & Violet era
- **Méga-Évolution** = current Mega Evolution era
- **Méga-Évolution – Chaos Ascendant** = Chaos Rising (ME04, sortie 22 mai 2026 FR; avant-première dès le 9 mai). Star du set: **Méga-Amphinobi-ex**. Coffret Dresseur d'Élite aux couleurs de Méga-Amphinobi.
- **151** = the popular Kanto set (very FR-collected)
- **Évolutions Prismatiques** = Prismatic Evolutions

## Community, buying, selling
| Term | Meaning |
|---|---|
| **Collec** | Collection. "10 ans de collec." |
| **Scalpeur / revendeur** | Scalper / reseller (negative). |
| **Drop / restock** | A release/restock event. Said in English. |
| **Notif / notifs de restock** | Restock notifications. Key CTA hook. |
| **Cardmarket** | The big EU marketplace. Reference for *la cote*. |
| **Se faire avoir / se faire arnaquer** | To get ripped off. |
| **Fake / carte fake** | Counterfeit. |
| **Sleeve / sleever** | To put cards in sleeves. "Sleeve tes cartes direct." |
| **Toploader / classeur** | Toploader / binder. |
| **Le hobby** | The hobby. |
| **Là-bas / chez eux** | When pointing to where you buy, prefer these over a bare "là" — "je le prends en scellé chez eux", not "…en scellé là". |

## Tone notes — sounds native vs sounds like an ad
- ✅ Native: *"franchement", "t'es au courant en premier", "ça vaut le coup", "fais gaffe", "perso", "de loin", "au pire"*
- ❌ Ad-smell: *"profitez de notre offre exceptionnelle", "le meilleur prix garanti", "n'attendez plus"*, excessive ❗❗❗, all-caps selling.
- Use *tu*, never *vous*. Contractions always. One idea per slide. Numbers/lists perform well.
- Anglicisms are correct here — *pull, chase, sealed, drop, restock, single, alt art* are how real collectors speak FR.
