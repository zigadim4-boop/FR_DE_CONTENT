# Content Learnings — MOVED

This file has been split so each persona writer loads only its own lane. Do **not** add rules here.

- **Universal craft rules** (hooks, slide order, promise→payoff, IvoryShard integration craft, depth,
  pacing) → [`content-rules-core.md`](content-rules-core.md) — every writer reads it.
- **Language-specific items** (diction, localized set/card names, banned promo words, spelling) →
  [`pack-de.md`](pack-de.md) (Lea + Mila) / [`pack-fr.md`](pack-fr.md) (Chloé), plus the lexicons
  [`slang-de.md`](slang-de.md) / [`slang-fr.md`](slang-fr.md).
- **Per-persona overrides** (a note for one persona only) → `persona-{lea|mila|chloe}.md` (created on demand).
- **Shared set facts** (dates, names, chase cards, counts) → [`verified-facts.md`](verified-facts.md).
- **Cross-persona / cross-day decisions** (IvoryShard repetition streak, peer-shop rotation, format
  distinctness, Lea ≠ Mila) → the orchestrator Steps in [`../workflows/daily_content.md`](../workflows/daily_content.md).

The `apply-feedback` skill routes new feedback to the right file above. No rule history is kept —
when a rule changes, the old wording is replaced, not archived.
